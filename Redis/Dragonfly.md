# Dragonfly

Redis on Flash = Redis协议兼容 + 热点数据在内存 + 全量数据在SSD。

混合存储（Hybrid Storage）功能目前仍处于**实验性（experimental）阶段**，其设计和实现正在快速迭代中。因此，**强烈不建议直接在生产环境中使用**，但非常适合用于技术验证和测试。
