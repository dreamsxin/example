# MoonBit
MoonBit是一个用于云计算和边缘计算的WebAssembly端到端编程语言工具链。 您可以在https://try.moonbitlang.com获得IDE环境，无需安装任何软件，也不依赖任何服务器。

```shell
/bin/bash -c "$(curl -fsSL https://www.moonbitlang.com/release/ubuntu_x86_64_moon_setup.sh)"
```
