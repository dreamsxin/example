# 最小公倍数

## 计算公式

最小公倍数 = 两个数之间的乘积/两个数之间的最大公因数

最大公因数可以通过辗转相除法得到。

这里证明一下为什么最小公倍数是两个数的乘积除于最大公因数。

假设一个数 A 和一个数 B 有最大公因数 x，则 A = a ∗ x，B = b ∗ x。其中数字 a 和数字 b 是互斥的。如果 a 和 b 不是互斥的话，可以得到更大的公因数。

那么易知得到 
A ∗ B = B ∗ A 
即 
A ∗ b ∗ x = B ∗ a ∗ x
公式中左右两端消除公因子 x 得到 
A ∗ b = B ∗ a 
因为 a 和 b 是互斥的，A 和 B是我们要求的最小公倍数的两个数。
