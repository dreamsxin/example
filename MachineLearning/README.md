# 机器学习

预测分类（离散型）和数值（连续型），得到近似解

分类问题：逻辑回归、朴素贝叶斯、支持向量机、随机森林

## 解决问题流程

1. 确定目标

- 业务需求
- 收集数据
- 特征工程：数据预处理、清理、整合、提取特征

图片的特征提取：RGB

2. 训练模型

- 定义模型：产生一个公式（系数未知）
- 定义损失函数：评估预测误差，得出相似度
- 优化算法：找到误差最小的系数

3. 模型评估

- 交叉验证
- 效果评估
