### 👑 双核心：Tailwind CSS 与 shadcn/ui

#### Tailwind CSS：AI 的「母语」

几乎所有的现代AI编程工具（如Cursor, v0, Copilot）在生成UI代码时，都表现出对Tailwind CSS的强烈偏好，这背后有深层技术原因：

*   **Token可预测性**：Tailwind的类名就是高度可预测的原子Token，LLM在处理训练数据时已经见过数百万次，非常擅长组合它们。
*   **上下文窗口效率**：样式与结构同在HTML中，避免了在JSX和CSS文件间的跳转，节省了宝贵的上下文Token。
*   **所见即所得**：生成的class直接贴在元素上，避免了AI在复杂的样式继承和优先级上出错，确保"一次生成就能运行"。
*   **免于命名**：AI绕开了传统CSS最难的部分——为class命名，直接应用样式，减轻了「认知负荷」。

市场趋势也印证了这一点，数据表明 **Tailwind使用量在2026年达到历史顶峰**，被开发者认为是"仍在增长的2026年技术栈"。

#### shadcn/ui：AI 时代的组件库标杆

如果说Tailwind是AI的样式"语言"，那shadcn/ui就是其"架构模式"。它不是npm包，而是一个CLI，能将组件**源码直接复制到你的项目中**，在AI时代大受欢迎。

*   **代码完全可控**：AI能直接读写和修改这些代码文件，因为没有任何抽象层或黑盒逻辑。
*   **实时且精准**：通过 **shadcn MCP Server**，AI助手能直接读取官方最新组件实现，避免了因训练数据滞后导致的错误代码，可将Bug减少90%，UI开发速度提高3倍。
*   **AI原生扩展**：可通过CLI安装专为AI应用设计的组件，如对话、代码块等，快速构建现代化的AI交互界面。

### 🚀 新浪潮：为AI而生的新一代框架

*   **HeroUI**：为AI Native应用打造的框架无关组件库，内置WCAG无障碍支持，由Framer Motion驱动动画，适合追求现代化审美的AI驱动项目。
*   **Strata**：宣称首个为AI辅助开发构建的设计系统，提供**AI可读文档** (llms.txt + MCP) 和**感知统一色彩系统**，可作为给AI的"设计说明书"。
*   **Tambo**：AI直接操控UI的SDK，你只需**注册组件**，AI根据用户意图决定渲染哪个组件及响应交互，适合构建下一代AI原生应用。
*   **Styleframe**：提供AI就绪的**类型安全 TypeScript CSS API**，类型系统可减少AI生成样式时的低级错误，适合TypeScript严格模式项目。

### 📋 框架选型速查表
